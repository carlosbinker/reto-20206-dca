CÓMO CREAR TU PROXY EN 5 MINUTOS

1. Ir a script.google.com → "Nuevo proyecto"
2. Borrar el contenido y pegar el código del archivo gas-proxy.js que se descarga junto con esta app
3. Menú Implementar → Nueva implementación
4. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier usuario
5. Copiar la URL generada y pegarla arriba
